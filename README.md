# DSA-Practice | Coding Interview Preparation

This repository contains my structured practice of Data Structures and Algorithms for coding interviews and placement preparation.

The goal of this repository is to strengthen problem-solving skills, improve algorithmic thinking, and build consistency through daily practice.

- Arrays

- Linked List

- Searching & Sorting
